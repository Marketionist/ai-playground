# ai-playground
AI playground with some useful code examples.

## Requirements
Python versions: 3.9+.

## Installation
1. Clone this repository:
```bash
git clone https://github.com/Marketionist/ai-playground.git
```

### Text to image
2. Create virtual environment and activate it:
```bash
python -m venv ai-playground/text-to-image
source ai-playground/text-to-image/bin/activate
```
3. Switch to text-to-image folder and install all dependencies:
```bash
cd ai-playground/text-to-image && pip install -r requirements.txt
```

## Running
### Text to image
```bash
PROMPT='pirate smiling with scimitar in the right hand' python main.py
```
This should generate a `image_result.png` image in the same folder.
